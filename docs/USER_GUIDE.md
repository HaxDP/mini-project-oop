# USER GUIDE

CinemaOOP - консольний застосунок для ролей адміністратора і касира. Меню просте: спочатку готуєш довідники, потім працюєш із сеансами, бронюванням і аналітикою.

Після запуску ти додаєш фільм і зал, створюєш сеанс, а далі переходиш до бронювання місця. Якщо дані коректні, система показує успіх і ідентифікатор операції. Якщо є помилка (наприклад, місце поза межами залу або замала сума оплати), виводиться зрозуміле повідомлення українською.

Збірка і запуск:

```bash
cmake -S . -B build
cmake --build build --config Debug
./build/Debug/cinema_app.exe
```

Дані зберігаються автоматично після операцій, тому можна безпечно закривати програму й продовжувати роботу пізніше.
