# Test Matrix

Матриця тестів зведена у компактному вигляді, щоб швидко бачити покриття логіки:

| ID | Рівень | Сценарій | Очікуваний результат |
|---|---|---|---|
| UT-01 | Unit | Створення фільму з порожньою назвою | Валідаційна помилка |
| UT-02 | Unit | Повторне бронювання того самого місця | Відмова, місце зайняте |
| UT-03 | Unit | Викуп із недостатньою оплатою | Відмова в оплаті |
| IT-01 | Integration | Успішне бронювання валідного місця | Бронювання створено |
| IT-02 | Integration | Бронювання місця поза межами залу | Валідаційна помилка |
| IT-03 | Integration | Викуп активного бронювання | Створений квиток, статус Purchased |
| IT-04 | Integration | Перезапуск після операцій | Стан відновлюється з файлів |
| IT-05 | Integration | Аналітичний запит за заповненістю | Повертаються очікувані записи |

У практичній роботі ця матриця використовується як короткий чек для regression-перевірки.
