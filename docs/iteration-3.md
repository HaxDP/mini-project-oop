# Iteration 3

Третя ітерація була присвячена якості. Після реалізації функціоналу головним питанням стало не «чи працює», а «наскільки надійно працює». Було оформлено тестову стратегію, тестову матрицю та додано CI-контроль із покриттям.

На виході отримано стабільний стан: основні позитивні й негативні сценарії покрито, а перевірка якості стала автоматичною частиною пайплайну.
